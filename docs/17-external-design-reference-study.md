# 외부 레퍼런스 기반 v4 디자인 정리

## 목적

이번 v4 디자인은 여러 금융앱 요소를 섞는 방식이 아니라, 하나의 제품 방향을 먼저 정한 뒤 화면 구성, 색감, 카드 밀도, 캘린더 표현을 같은 규칙으로 맞추는 것을 목표로 한다.

우리가 잡을 방향은 **밝은 개인 금융 코치형 소비 캘린더**다. 금융앱처럼 신뢰감 있고, 캘린더앱처럼 한눈에 읽히며, AI 서비스처럼 보이기보다는 사용자의 소비 결정을 도와주는 코치처럼 느껴져야 한다.

## 참고한 공개 레퍼런스

### 1. Toss

참고 URL: https://toss.im/

토스는 홈, 소비, 계좌, 투자 등 금융 정보를 한곳에 모으는 구조가 강하다. 특히 소비 화면에서 월별·일자별 수입/지출을 보여주는 패턴은 우리 서비스의 핵심인 소비 캘린더와 직접 연결된다.

적용점:

- 첫 화면은 "AI 대시보드"가 아니라 "내 돈 상태"가 먼저 보여야 한다.
- 숫자는 크게, 설명은 짧게 둔다.
- 소비 분석은 별도 리포트 페이지로 숨기기보다 홈 카드와 캘린더 안에 자연스럽게 녹인다.

### 2. TravelWallet

참고 URL: https://www.travel-wallet.com/

트래블월렛은 밝은 배경, 큰 지갑 카드, 명확한 충전/송금 액션처럼 해야 할 행동이 바로 보이는 구조를 쓴다. 복잡한 금융 정보를 단순한 카드와 액션으로 정리하는 점이 좋다.

적용점:

- 메인 카드 안에서 "이번 달 소비", "남은 예산", "저축 예상"을 바로 확인하게 한다.
- 주요 행동은 4개 이하로 제한한다.
- 파란색을 주 색으로 쓰되, 화면 전체를 파랗게 만들지는 않는다.

### 3. Notion Calendar

참고 URL: https://www.notion.com/product/calendar

노션 캘린더는 여러 일정을 한 화면에서 정리하고, 다음 일정을 빠르게 파악하는 데 집중한다. 우리 서비스에서는 일정 대신 소비와 구독 결제일을 다룬다.

적용점:

- 캘린더는 장식이 아니라 제품의 중심 기능으로 보여야 한다.
- 날짜 셀마다 모든 정보를 욱여넣기보다 색, 점, 짧은 금액으로 상태를 요약한다.
- 선택된 날짜 아래에 "왜 중요한 날인지"를 바로 설명한다.

### 4. Nielsen Norman Group 모바일 내비게이션 자료

참고 URL: https://www.nngroup.com/articles/mobile-navigation-patterns/

모바일 앱은 하단 탭처럼 반복 접근이 쉬운 내비게이션이 중요하다. 우리 서비스도 홈, 캘린더, 추가, 코치, 설정을 일관되게 두면 심사위원이 짧은 시간 안에 화면을 이해하기 쉽다.

적용점:

- 하단 탭은 항상 고정한다.
- 가운데 "추가" 버튼만 강조하고 나머지는 조용하게 둔다.
- 탭 이름은 기능 설명이 아니라 사용자가 찾는 장소 이름으로 쓴다.

### 5. YNAB

참고 URL: https://www.ynab.com/

YNAB은 예산을 단순 기록이 아니라 행동 기준으로 만든다. 우리 서비스의 AI도 "분석 결과"를 보여주는 데서 끝나면 약하고, 오늘부터 조정할 수 있는 행동으로 이어져야 한다.

적용점:

- "카페를 줄이세요"보다 "이번 주 카페 2회만 유지하면 12,000원 절약"처럼 행동 단위를 제시한다.
- 목표 소비액, 목표 저축액, 남은 일수를 한 문장으로 연결한다.
- 사용자가 실패했다고 느끼지 않도록 경고보다 조정안을 먼저 보여준다.

### 6. 사용자가 제공한 금융앱 캡처

참고 이미지: 우리은행, 카카오뱅크, 카카오페이, 우리카드, 트래블월렛 계열 캡처

공통적으로 확인한 점은 모바일 금융앱의 첫 화면이 "큰 숫자 카드 + 빠른 액션 + 피드/내역 + 하단 탭"으로 구성된다는 것이다. 다만 이전 v2처럼 캘린더만 어두운 톤으로 두면 화면이 따로 노는 느낌이 강했다.

적용점:

- 전체 앱은 밝은 톤으로 통일한다.
- 캘린더도 밝은 카드 안에 넣고, 상태 색만 다르게 쓴다.
- 광고 배너나 캐릭터는 쓰지 않고 개인화 코칭 카드로 대체한다.

## v4 디자인 원칙

### 색감

| 역할 | 색상 | 사용 |
| --- | --- | --- |
| 배경 | `#F5F7FA` | 전체 앱 배경 |
| 표면 | `#FFFFFF` | 카드, 바텀 내비게이션 |
| 본문 | `#16191F` | 주요 숫자, 제목 |
| 보조 텍스트 | `#697281` | 설명, 날짜 보조 정보 |
| 메인 블루 | `#2478FF` | 주요 액션, 선택 상태 |
| 소프트 블루 | `#EAF2FF` | 코치 카드, 선택 날짜 |
| 세이빙 민트 | `#00A878` | 절약, 목표 달성 |
| 경고 코랄 | `#FF665A` | 초과 소비, 위험 신호 |
| 구독 앰버 | `#F6A623` | 고정비, 구독 결제 |

### 배치

1. 상단: 서비스명, 날짜, 알림
2. 첫 카드: 이번 달 소비와 목표 진행률
3. 코치 문장: 지금 조정하면 바뀌는 행동 1개
4. 빠른 액션: 내역 추가, 파일 연결, 목표, 코치
5. 밝은 소비 캘린더: 날짜별 소비 강도와 구독일
6. 오늘의 조정: 미션 2개
7. 구독/최근 내역: 심사위원이 기능 범위를 바로 이해하는 보조 카드
8. 하단 탭: 홈, 캘린더, 추가, 코치, 설정

### 버릴 것

- 전체 화면을 채우는 보라/파랑 그라데이션
- 금융앱 광고 배너처럼 보이는 큰 홍보 카드
- 앱마다 가져온 요소가 섞인 혼합 톤
- "AI 리포트" 같은 기능명 중심 문구
- 달력만 다크모드로 분리하는 방식

## 새 스크린샷 목표

v4 화면은 "실제 배포 가능한 모바일 금융앱 홈"처럼 보여야 한다. 단, 금융앱을 그대로 따라 하는 것이 아니라 **소비 캘린더가 중심인 개인 금융 코치**로 차별화한다.
