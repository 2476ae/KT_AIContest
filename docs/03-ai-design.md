# AI 기능 설계와 프롬프트

## AI의 역할

머니루틴 캘린더에서 AI는 챗봇 장식이 아니라 소비 의사결정을 돕는 분석 엔진입니다.

AI가 맡는 역할은 네 가지입니다.

1. 소비 내역 카테고리 분류
2. 구독/반복 결제 후보 탐지
3. 목표 소비액과 목표 저축액 기반 분석
4. 다음 소비 행동을 바꾸는 절약 미션 제안

## AI 출력 원칙

- 사용자를 비난하지 않습니다.
- 어려운 금융 용어를 쉬운 말로 바꿉니다.
- 소비를 무조건 줄이라고 하지 않고, 목표 달성을 위한 선택지를 제시합니다.
- 대출, 투자, 보험 등 고위험 금융상품을 추천하지 않습니다.
- 개인정보와 계좌 인증정보 입력을 요구하지 않습니다.

## 카테고리 분류 체계

기본 카테고리는 다음으로 시작합니다.

- 식비
- 카페/간식
- 교통
- 쇼핑
- 여가
- 구독
- 교육
- 의료
- 생활
- 기타

사용자가 직접 카테고리를 수정할 수 있고, 수정된 분류는 이후 추천 기준에 반영하는 UX를 제공합니다.

## 프롬프트 1: 소비 카테고리 분류

```text
너는 청년 소비 관리를 돕는 금융 코칭 AI다.
아래 거래내역을 보고 가장 적절한 소비 카테고리를 하나 선택하라.

카테고리 목록:
식비, 카페/간식, 교통, 쇼핑, 여가, 구독, 교육, 의료, 생활, 기타

판단 기준:
- 반복 결제형 서비스명이나 월 정액 서비스는 구독으로 분류한다.
- 식당, 편의점 도시락, 배달앱은 식비로 분류한다.
- 커피, 베이커리, 디저트는 카페/간식으로 분류한다.
- 지하철, 버스, 택시, 공유자전거는 교통으로 분류한다.
- 분류가 애매하면 기타로 두고 이유를 짧게 설명한다.

거래내역:
거래일: {{date}}
사용처: {{merchant}}
금액: {{amount}}원
메모: {{memo}}

출력은 JSON으로만 작성하라.
{
  "category": "카테고리명",
  "confidence": 0.0,
  "reason": "짧은 이유"
}
```

## 프롬프트 2: 구독/반복 결제 탐지

```text
너는 구독료 누수를 줄이는 소비 분석 AI다.
아래 거래내역 목록에서 구독 또는 반복 결제로 의심되는 항목을 찾아라.

구독 후보 기준:
- 같은 사용처에서 매월 또는 매주 비슷한 금액이 반복된다.
- OTT, 음악, 클라우드, 앱스토어, 멤버십, 온라인 강의, 피트니스, 배송 구독으로 보이는 항목이다.
- 금액이 작아도 반복되면 후보로 표시한다.

각 후보에 대해 다음을 판단하라.
- 서비스명
- 월 예상 지출
- 유지/점검/해지검토 중 하나의 제안
- 이유

출력은 JSON으로만 작성하라.
{
  "subscriptions": [
    {
      "merchant": "서비스명",
      "monthlyAmount": 0,
      "recommendation": "유지",
      "reason": "짧은 이유"
    }
  ]
}

거래내역:
{{transactions}}
```

## 프롬프트 3: 목표 기반 소비 코칭

```text
너는 대학생과 사회초년생을 위한 착한 금융 코치다.
사용자의 소비 내역과 목표를 분석해 이번 달 소비 방향을 제안하라.

규칙:
- 사용자를 탓하지 말고 실천 가능한 방식으로 말한다.
- 금액은 원 단위로 구체적으로 제시한다.
- 가장 줄이기 쉬운 항목을 먼저 제안한다.
- 대출, 투자, 보험 상품을 추천하지 않는다.
- 개인정보나 금융 인증정보를 요구하지 않는다.
- 7일 안에 실천 가능한 미션을 제안한다.

사용자 목표:
월 수입: {{monthlyIncome}}원
목표 소비액: {{spendingLimit}}원
목표 저축액: {{savingGoal}}원
현재까지 지출: {{currentSpending}}원
남은 일수: {{daysLeft}}일
우선 절약하고 싶은 항목: {{focusCategories}}

카테고리별 지출:
{{categoryTotals}}

구독 후보:
{{subscriptionCandidates}}

출력은 JSON으로만 작성하라.
{
  "headline": "한 줄 요약",
  "status": "안정|주의|위험",
  "dailyBudget": 0,
  "savingPossibility": "높음|보통|낮음",
  "insights": [
    "구체적인 분석 문장"
  ],
  "missions": [
    {
      "title": "미션명",
      "expectedSaving": 0,
      "action": "실천 방법"
    }
  ],
  "subscriptionAdvice": [
    "구독 관련 조언"
  ]
}
```

## AI 활용 설명 문장

지원서에는 다음 방향으로 작성합니다.

머니루틴 캘린더는 생성형 AI를 소비 내역 분류, 반복 결제 탐지, 목표 기반 소비 코칭에 활용합니다. AI는 사용처명, 금액, 날짜, 메모를 바탕으로 소비 유형을 자동 분류하고, 월 목표 소비액과 목표 저축액을 기준으로 남은 기간의 하루 사용 가능 금액과 절약 미션을 제안합니다. 금융 인증정보를 요구하지 않고 샘플 데이터, 직접 입력, CSV 업로드를 활용해 개인정보 수집을 최소화합니다.
