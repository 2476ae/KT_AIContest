제출 URL 기준: `https://kt-ai-contest.vercel.app/`

## 0. 기준 출처와 적용 방식

이 평가표는 공식 공모전 참가 안내 페이지와 저장소에 정리된 공모전 연결 문서, 제출 런북을 함께 기준으로 만든다.

기준이 되는 핵심 정보:

- 공모전: K-AI Contents Award
- 공통 주제: 우리 곁의 착한 AI, 일상을 바꾸는 상상력
- 지원 트랙: Track B. 솔루션 부문
- Track B 정의: 일상의 문제 해결을 위한 AI 기술 활용 솔루션 제안
- Track B 제출물: 구동 가능한 웹페이지 링크(URL)
- 목표 상: AX 핀테크상
- Track B 경제/금융 예시 연결: 스마트한 경제 활동, 슬기로운 소비생활, 새나가는 구독료 관리
- 공식/기획 기준 배점: 문제해결력과 창의성 20점, 기술 구현 완성도 25점, UX 및 디자인 심미성 25점, 실용성 및 발전가능성 30점

공식 안내에서 추가로 확인한 제출/유의사항:

- 서비스 설명에 AI 적용 기술, 개발 언어 및 라이브러리, 핵심 프롬프트를 포함해야 한다.
- 오픈소스 소프트웨어 사용 시 관련 라이선스와 사용 범위를 명시해야 한다.
- 제출물은 법적, 보안, 윤리, 개인정보 관련 이슈를 포함하지 않아야 한다.
- 주최 측 요청 시 소스 코드 제출이 필요할 수 있다.
- 제출 후 지정된 기간에는 멘토링 참석이 가능해야 한다.

## 8. 심사위원 시나리오 기반 체크리스트

### 0~10초: 첫인상

- [ ] 인트로가 자연스럽고 과하지 않다.
- [ ] 홈 화면으로 시작한다.
- [ ] 앱 이름과 목적이 즉시 보인다.
- [ ] 샘플 데이터 없이도 소비 캘린더/목표 코치 앱임을 알 수 있다.

### 10~40초: 샘플 데이터 로드

- [ ] 샘플 데이터 로드 버튼이 보인다.
- [ ] 클릭 후 월 지출, 목표 진행률, 하루 예산, 정기 결제가 채워진다.
- [ ] 로딩/성공 피드백이 있다.

### 40~80초: 캘린더 탐색

- [ ] 날짜별 지출 금액이 직관적이다.
- [ ] 초과/적정/정기 결제 필터가 작동한다.
- [ ] 선택 날짜의 거래 목록이 보인다.
- [ ] 날짜 아래 이상한 텍스트가 없다.

### 80~120초: 목표 수정

- [ ] 월수입, 목표 소비액, 목표 저축액을 수정할 수 있다.
- [ ] 저장 후 홈/캘린더/AI 코치 기준이 갱신된다.
- [ ] 목표 초과 상황도 현실적으로 계산된다.

### 120~160초: 소비 추가

- [ ] 직접 입력이 가능하다.
- [ ] 자동 분류 또는 수동 분류를 선택할 수 있다.
- [ ] 저장 후 캘린더와 최근 내역에 반영된다.

### 160~210초: AI 코치 확인

- [ ] AI 코치 화면의 문장이 짧고 행동 중심이다.
- [ ] 오늘 사용할 수 있는 금액의 의미가 명확하다.
- [ ] 분야별 소비 계획이 보인다.
- [ ] 정기 결제 점검이 과장 없이 제안된다.
- [ ] OpenAI 분석 업데이트는 명시 버튼에서만 실행된다.

### 210~240초: 설정/신뢰 확인

- [ ] 금융 인증정보를 수집하지 않는다는 안내가 있다.
- [ ] 실시간 반영 준비 상태가 보인다.
- [ ] CSV 내보내기/초기화/샘플 로드가 안정적이다.
