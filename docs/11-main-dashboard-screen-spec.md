# 메인 대시보드 상세 시안 명세

## 목적

메인 대시보드는 머니루틴 캘린더의 첫인상을 결정하는 화면입니다. 이 화면의 목표는 설명 없이도 다음 세 가지를 즉시 전달하는 것입니다.

1. 이 앱은 소비를 달력으로 읽는 서비스입니다.
2. AI는 소비 목표 달성을 위한 판단과 다음 행동을 제안합니다.
3. 구독과 반복 지출을 별도로 점검해 새는 돈을 줄입니다.

## 화면 한 줄 목표

> "이번 달 돈이 어디서 새고 있는지, 오늘부터 무엇을 줄이면 되는지 바로 알 수 있다."

## 데스크톱 레이아웃

권장 기준은 1440px 너비입니다.

```text
┌──────────────────────────────────────────────────────────────┐
│ 상단 앱 바: 서비스명 / 탭 / 샘플 소비 버튼 / 개인정보 안내     │
├──────────────────────────────────────────────────────────────┤
│ 헤더: 6월 소비 상태 / 한 줄 설명 / 목표 설정 요약              │
├──────────────────────────────────────────────────────────────┤
│ 요약 카드 4개: 총 지출 / 목표 사용률 / 하루 예산 / 저축 가능성 │
├───────────────────────────────────────┬──────────────────────┤
│ 소비 캘린더                            │ AI 오늘의 판단        │
│ 날짜별 금액, 상태, 구독 뱃지            │ 결론, 이유, 미션       │
├───────────────────────┬───────────────┴──────────────────────┤
│ 구독/반복 결제 후보    │ 카테고리별 소비 흐름                  │
└───────────────────────┴──────────────────────────────────────┘
```

### 권장 비율

| 영역 | 비율 | 이유 |
| --- | --- | --- |
| 캘린더 | 2fr | 서비스의 차별점이므로 가장 넓게 배치합니다. |
| AI 판단 | 1fr | 결론과 미션을 빠르게 읽는 보조 패널입니다. |
| 구독 후보 | 1fr | 공모전 경제/금융 예시와 직접 연결됩니다. |
| 카테고리 흐름 | 2fr | 소비 분류 기능의 결과를 보여줍니다. |

## 모바일 레이아웃

모바일은 390px 너비를 기준으로 합니다.

순서:

1. 앱 바
2. AI 한 줄 판단
3. 요약 카드 2열
4. 소비 캘린더
5. 선택 날짜 상세
6. 구독 후보
7. 카테고리 흐름
8. 하단 탭

모바일에서는 캘린더를 먼저 보이고 싶더라도, AI 한 줄 판단을 더 위에 둡니다. 작은 화면에서는 사용자가 "지금 무엇을 해야 하는지"를 먼저 알아야 하기 때문입니다.

## 소비 캘린더 상세

### 날짜 셀 필수 요소

- 날짜 숫자
- 하루 총 지출
- 소비 강도 배경
- 최대 2개 카테고리 점
- 구독 결제일 뱃지
- 상태 라벨

### 상태 예시

| 날짜 | 지출 | 상태 | 표시 |
| --- | ---: | --- | --- |
| 6월 4일 | 25,900원 | 구독 + 교통 | 구독 뱃지, 파란 점 |
| 6월 10일 | 52,000원 | 과소비 | 초과 라벨 |
| 6월 21일 | 1,000원 | 안정 | 낮은 강도 |
| 6월 30일 | 0원 | 무지출 | 0원 라벨 |

### 선택 날짜 패널

선택 날짜를 클릭하면 캘린더 아래 또는 옆에 짧은 상세를 보여줍니다.

구성:

- 날짜
- 총 지출
- 거래 목록 3~5개
- AI 한 줄 코멘트

예시:

> 6월 10일은 쇼핑과 카페 지출이 겹쳐 하루 권장 예산보다 높았어요.

## AI 오늘의 판단 카드

### 구조

1. 상태 라벨
2. 한 줄 결론
3. 핵심 숫자
4. 이유 2개
5. 이번 주 미션 3개

### 실제 문구

이유:

- 배달 지출은 주말 저녁에 반복되는 경향이 있습니다.

미션:

1. 카페 2회 줄이기: 12,000원 절약
2. 배달 하루 쉬기: 18,000원 절약
3. 음악 구독 점검: 10,900원 절약

## 구독/반복 결제 후보

첫 화면에서 반드시 보여야 합니다. 공모전 예시인 "새나가는 구독료 잡기"와 직접 연결되는 부분입니다.

### 표시 항목

- 서비스명
- 월 결제 금액
- 결제일
- 제안 라벨

### 예시

| 서비스 | 금액 | 제안 |
| --- | ---: | --- |
| 넷플릭스 | 13,500원 | 유지 |
| 유튜브프리미엄 | 14,900원 | 점검 |
| 스포티파이 | 10,900원 | 해지 검토 |
| 노션AI | 16,500원 | 점검 |

## 카테고리별 소비 흐름

### 목적

AI 자동 분류 기능이 실제로 작동하는 것처럼 보여주는 영역입니다.

### 표시 방식

- 카테고리별 총액
- 전체 지출 대비 비중
- 목표 대비 상태
- 한 줄 인사이트

## 시각적 기준

### 분위기

- 신뢰감 있는 금융 도구
- 매일 열어볼 수 있는 생활 앱
- 캘린더가 중심인 작업 화면

### 피해야 할 인상

- AI 랜딩페이지
- 예쁜 카드 모음
- 은행 앱 흉내
- 기업용 분석 대시보드

## 구현자가 지켜야 할 핵심

- 첫 화면에서 실제 앱처럼 보여야 합니다.
- 캘린더가 가장 넓고 강한 시각 요소여야 합니다.
- AI 판단은 길지 않아야 합니다.
- 구독 후보는 첫 화면 하단 또는 우측에 반드시 보여야 합니다.
- 모바일에서는 텍스트가 캘린더 셀 안에서 겹치지 않아야 합니다.
