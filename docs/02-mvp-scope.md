# MVP 범위와 화면 흐름

## MVP 원칙

이번 제출작은 은행 연동을 약속하는 앱이 아니라, **개인정보 수집을 최소화한 완성형 AI 소비 캘린더**입니다.

핵심은 다음 세 가지입니다.

1. 소비 데이터를 쉽게 넣을 수 있어야 합니다.
2. AI가 소비 패턴을 이해 가능한 언어로 분석해야 합니다.
3. 목표 소비액과 목표 저축액을 기준으로 다음 행동을 제안해야 합니다.

## 포함 기능

- 샘플 소비 데이터 불러오기
- 소비 내역 직접 입력
- CSV 업로드
- 소비 유형 직접 수정
- AI 소비 카테고리 자동 분류
- 월간 소비 캘린더
- 과소비일, 무지출일, 구독 결제일 표시
- 월 목표 소비액 설정
- 월 목표 저축액 설정
- 남은 기간 하루 사용 가능 금액 계산
- AI 절약 리포트 생성
- 개인정보/금융 인증정보 미수집 안내

## 제외 기능

- 실제 은행 앱 연동
- 카드사 자동 연동
- 금융 인증서 로그인
- 계좌 잔액 실시간 조회
- 구독 서비스 자동 해지
- 투자 상품 추천
- 대출, 보험, 고위험 금융상품 추천

## 3분 체험 시나리오

심사위원이 웹앱에 들어온 뒤 3분 안에 다음 흐름을 경험하게 합니다.

1. 첫 화면에서 "샘플 데이터로 체험하기"를 누릅니다.
2. 월 목표 소비액 750,000원, 목표 저축액 300,000원을 입력합니다.
3. 앱이 소비 내역을 자동 분류하고 월간 캘린더를 표시합니다.
4. 사용자는 구독 결제일과 과소비일을 확인합니다.
5. AI 리포트에서 "가장 줄이기 쉬운 항목", "남은 기간 하루 예산", "다음 주 절약 미션"을 확인합니다.

## 화면 구성

### 1. 대시보드

목적: 현재 소비 상태를 한눈에 보여줍니다.

주요 요소:

- 이번 달 총 지출
- 목표 소비액 대비 사용률
- 목표 저축 달성 가능성
- 남은 기간 하루 예산
- AI 한 줄 코멘트
- 샘플 데이터 불러오기 버튼

### 2. 소비 캘린더

목적: 날짜별 소비 흐름과 반복 지출을 시각화합니다.

주요 요소:

- 월간 달력
- 날짜별 소비 총액
- 과소비일 표시
- 무지출일 표시
- 구독 결제일 뱃지
- 날짜 클릭 시 해당 일의 거래 목록

### 3. 소비 내역

목적: 소비 데이터를 추가하고 수정합니다.

주요 요소:

- 직접 입력 폼
- CSV 업로드
- 카테고리 필터
- 거래 목록
- 카테고리 수동 수정
- 반복 지출 여부 표시

### 4. 목표 설정

목적: AI 분석 기준이 되는 목표를 입력합니다.

주요 요소:

- 월 수입
- 월 목표 소비액
- 월 목표 저축액
- 카테고리별 예산
- 우선 절약하고 싶은 항목

### 5. AI 리포트

목적: 사용자의 소비 패턴을 쉬운 말로 분석하고 행동 전략을 제안합니다.

주요 요소:

- 이번 달 소비 요약
- 카테고리별 위험도
- 반복 결제/구독 후보
- 절약 가능 금액 추정
- 남은 기간 하루 예산
- 다음 주 실천 미션 3개

## 데이터 모델 초안

### Transaction

| 필드 | 설명 | 예시 |
| --- | --- | --- |
| id | 거래 ID | tx-001 |
| date | 거래일 | 2026-06-03 |
| merchant | 사용처 | 스타벅스 |
| amount | 지출 금액 | 5800 |
| memo | 사용자 메모 | 등교 전 커피 |
| category | 소비 유형 | 카페 |
| paymentType | 결제 방식 | 카드 |
| isSubscription | 구독 여부 | false |

### Goal

| 필드 | 설명 | 예시 |
| --- | --- | --- |
| monthlyIncome | 월 수입 | 1200000 |
| spendingLimit | 목표 소비액 | 750000 |
| savingGoal | 목표 저축액 | 300000 |
| focusCategories | 우선 절약 항목 | 카페, 배달, 구독 |

### AIReport

| 필드 | 설명 |
| --- | --- |
| summary | 전체 소비 요약 |
| categoryInsights | 카테고리별 분석 |
| subscriptionRisks | 구독/반복 결제 분석 |
| dailyBudget | 남은 기간 하루 예산 |
| savingPlan | 저축 목표 달성 전략 |
| missions | 다음 주 실천 미션 |

## 완료 기준

MVP는 다음 조건을 만족하면 제출 가능한 완성형으로 봅니다.

- 샘플 데이터만으로 전체 흐름을 체험할 수 있습니다.
- 직접 입력한 거래가 캘린더와 리포트에 반영됩니다.
- AI 분류 또는 AI 분석 결과가 화면에 표시됩니다.
- 목표를 바꾸면 리포트 내용도 달라집니다.
- 모바일 화면에서도 주요 흐름이 깨지지 않습니다.
- 금융 인증정보를 요구하지 않는다는 안내가 명확합니다.
